# 航海同盟：秘寶大爭奪 2 (Pirate Battle 2)

這是一個基於 **Ren'Py 8** 引擎開發的視覺小說與團康互動遊戲。
本專案改編自同盟海盜尋寶的互動故事，融合了精彩的對話劇情與 **六款精心設計的白板互動小遊戲**，非常適合團隊建立（Team Building）、營隊或派對團康活動。

---

## 📖 故事背景

你們是十艘正共同航行的海盜船。表面上，大家維持著共同尋找傳說祕寶的合作同盟關係……但在背地裡卻是暗潮洶湧，每位船長都各懷鬼胎。

當你們航行了大半年，終於在一座未標記在普通海圖上的小島前方相遇，而祕寶的位置正巧與此處重合。然而，隨著登島與探索的深入，這座島上的原住民守衛、劇烈塌陷的礦場、神祕的紫霧山洞以及嚴密的監牢，都考驗著海盜們的貪婪與智慧。究竟誰能帶走最多的寶藏，成為海盜之王？

---

## 🎨 遊戲特色

* **角色立繪與視覺特效**：結合表情變更、畫面震動（`hpunch` / `vpunch`）效果，打造動態的對話演出。
* **環境氛圍營造**：利用程式碼實現「半透明紫色濃霧濾鏡」與「洞口動態呼吸金光」效果，不需額外美術素材即可擁有精緻的氛圍。
* **整合式計時器系統**：每款遊戲皆內建專屬的倒數計時器畫面，支援：
  - 自訂倒數時間（30秒、60秒、90秒、120秒，或手動增減 10 秒）。
  - 計時暫停、重設與倒數結束通知。
  - 清晰、大字體的規則閱讀區。

## 🎮 六大團康小遊戲規則

本遊戲的設計核心在於 **「小白板秘密寫值，翻開結算」**。每款遊戲均有不同的博弈與策略機制：

### 1. 遊戲一：逃出迷霧 (平均乘 0.8)
每支小隊預測航行距離，秘密寫下 `1 ~ 100` 的整數。真實距離為所有隊伍平均值 $\times 0.8$。最接近真實距離的前三名小隊獲得 `3分、2分、1分`。

### 2. 遊戲二：採椰子大賽 (最大唯一數)
每支小隊秘密寫下 `1 ~ 15` 之間的一個整數（想搶的椰子）：
1. **撞車摔落（淘汰）**：數字若與其他隊伍重複，則該數字的所有小隊出局（得 0 分）。
2. **高樹採收（得分）**：在剩下的「唯一數字」中排名，最大的前三名依序獲得 `3分、2分、1分`。

### 3. 遊戲三：我有一個淘金夢 (限制容量博弈)
有 A、B、C 三座礦場，小隊秘密寫下代號選擇前往。
* **A 礦場**：金礦，價值 3 分（最多容納 2 隊）
* **B 礦場**：銀礦，價值 2 分（最多容納 3 隊）
* **C 礦場**：銅礦，價值 1 分（最多容納 6 隊）
* **崩塌機制**：若前往該礦場的隊伍數 **超過** 上限，礦場將會崩塌，裡面的隊伍全得 `0 分`。未崩塌礦場的隊伍獲得對應分數。

### 4. 遊戲四：越獄行動 (總和博弈)
秘密寫下 `0 ~ 10` 的整數代表逃獄速度（動靜）：
1. **神不知鬼不覺（總和 ≤ 60）**：動靜小於等於 60 代表沒被守衛發現。速度最快（數字最大）的前三名獲得 `3分、2分、1分`。
2. **警鈴大作（總和 > 60）**：動靜超過 60 被守衛發現。速度最慢（數字最小）的三個小隊及時退回，趁亂從後門溜出，按數字由小到大獲得 `3分、2分、1分`。

### 5. 遊戲五：毒氣中的橫財 (時間差判定)
秘密寫下 `0 ~ 300` 之間的秒數代表進入山洞的時間。由小到大結算：
1. **第一位抵達者（啟動機關）**：數字最小的隊伍取走毒瘴水晶，獲得 `時間 / 10` 的分數，並啟動抽風機（需 60 秒排毒）。
2. **後續抵達者（搶奪寶物）**：在毒氣散去（即 `啟動秒數 + 60`）後最先進入的前三名，獲得 `3分、2分、1分`。
3. **提早進入（中毒淘汰）**：除第一位外，在毒氣散去前就進入的小隊皆得 `0 分`。

### 6. 遊戲六：通往天國的倒數計時 (時間軸霸佔)
小隊填入 `0 ~ 600` 的數字 $t_0$ 代表進入財寶庫的時間。進入時會把裡面的人踢出：
* 你能搜刮寶物的時間為你進入到下一個人 $t_1$ 進來為止（獲得 $t_1 - t_0$ 份財寶，最後一個進入者搜刮到第 600 秒為止）。
* 若 $k$ 個隊伍在同一秒 $t_0$ 同時進入，則平分該段時間的財寶：$\frac{t_1 - t_0}{k}$。
* 搜刮財寶總數最多的前三名，依序獲得 `3分、2分、1分`。

## 📂 專案結構

* **`game/script.rpy`**：遊戲主線劇情對話、場景轉換、立繪控制與轉場音效。
* **`game/minigame.rpy`**：定義六個小遊戲的說明 UI 畫面 (`screen`)、計時器核心邏輯與互動 label。
* **`game/images/`**：存放背景圖、角色立繪素材及自動生成的金光素材 `golden_glow.png`。
* **`.gitignore`**：已設定自動忽略 `saves/`、`cache/`、`*.rpyc` 等 Ren'Py 本地生成檔案，確保 Git 儲存庫乾淨。

## 🚀 如何執行

1. 下載並安裝 [Ren'Py SDK](https://www.renpy.org/)。
2. 將本專案資料夾（`pirate battle 2`）放入您的 Ren'Py 專案目錄中。
3. 開啟 Ren'Py Launcher，在專案列表中選擇 **「pirate battle 2」**。
4. 點擊 **「啟動專案 (Launch Project)」** 即可開始遊玩！
